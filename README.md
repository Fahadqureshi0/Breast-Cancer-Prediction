# 🩺 Breast Cancer Prediction

A Machine Learning project that predicts whether a breast tumor is **Benign** or **Malignant** using the **Breast Cancer Wisconsin Dataset**.

---

## 📌 Project Overview

This project uses a **LogistisRegression** to predict breast cancer based on 30 medical features extracted from cell nuclei in breast mass images.

The objective is to assist in the early detection of breast cancer through machine learning classification.

## 📊 Dataset

- **Dataset:** Breast Cancer Wisconsin Dataset

- **0 → Benign**
- **1 → Malignant**
